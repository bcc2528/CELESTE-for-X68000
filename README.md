# CELESTE Classic (PICO-8) for X68000
![CELESTE for X68000](Screenshots/title.png?raw=true)

PICO-8用に作られたプラットフォームアクションゲーム「CELESTE Classic」をパーソナルワークステーション「X68000」に移植したものです。
過去にFM TOWNSにも移植していたのでその時のソースコードを流用し、TOWNS版がすべてスプライトで描画(背景点滅のみビットマップ面)していたのに対し、X68000版は背景多重スクロールにグラフィック面3枚、キャラクターと文字表示にスプライト面、地形の表示にBG面1枚を使用しています。

要68000 10MHz・メモリ1MB、つまり全X68000シリーズ機・メモリを増設せずに起動可能です。ゲームファイル単体の他、そのままブート可能なXDF形式のフロッピーイメージファイルも用意しています。

## 操作方法:
| キーボード| ゲームパッド| アクション     |
|----------|-------------|---------------|
|Zキー      |Aボタン     |ジャンプ        |
|Xキー      |Bボタン     |ダッシュ(空中化)|
|矢印キー   |十字キー    | 移動           |
|ESC       |            |ポーズ/ゲーム終了|

ダッシュは空中で使用可能、十字キー + ダッシュで十字キーを押している方向に飛んでいく。空中に浮遊している風船やフルーツを取ると地面に着地しなくてもダッシュ回数が回復。最初は一度しか使えないが、中間地点のオーブを取ってパワーアップすることで二回まで使用可能となる。
ゲーム中にESCキーを押すとポーズ、その状態でダッシュを押すとタイトル画面に戻る。更にタイトル画面でESCキーを押すとゲームが終了する。
